    LLMFirewall

    GitHub ↗

    Security proxy for tool-using LLM agents that detects prompt-injection attempts before high-risk tool calls. MCP is the protocol agents use to call external tools; LLMFirewall sits between agent and tools enforcing taint-aware trust levels, capability gating, heuristic detectors, and lightweight semantic-drift scoring.

    4,200+ eval cases 100%→16% ASR (vs. no-filter) sub-ms policy checks 783 tests Python · FastAPI · MCP

    C++20 Matching Engine

    GitHub ↗

    Exchange-style matching engine with deterministic price-time priority, FIFO fairness, and reproducible benchmark harnesses. Optimized with slab allocators, cache-conscious order layout, branch reduction, and CPU pinning. Includes Binance US market data connector, TAQ replay, and crash recovery.

    20M+ msgs/sec p50 = 0.04 µs p99 ≈ 1 µs C++20 · Python · Docker · Streamlit

    Synthetic local benchmark — hardware spec and harness in repo README.
