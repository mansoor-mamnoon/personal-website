# Portfolio Website — Mansoor Mamnoon

## Project context
This is a personal portfolio website for a UC Berkeley EECS Honors student (Class of 2027) applying to new grad roles in distributed systems, infrastructure, HFT, and quant engineering.

## Target audience
Recruiters and hiring managers at FAANG, HFT firms (Jane Street, Citadel, Two Sigma), quant funds, and unicorn startups (Databricks, Stripe, etc).

## Tech stack
Plain HTML, CSS (style.css), and vanilla JS (main.js). No framework. Multi-page site: index.html, projects.html, experience.html, about.html, notes.html, resume.html.

## Goal
Every change should make the site more effective at getting a recruiter to spend more than 8 seconds on it and forward the candidate to a hiring manager.

## Key constraint
Do not change the visual aesthetic — dark theme, Newsreader/Geist fonts, existing color palette. Only change structure, content, and information hierarchy.
